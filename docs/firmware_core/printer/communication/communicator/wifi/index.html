Module wifi

Source
Expand description

This module provides the functionality for WiFi communication in the microcontroller.

It defines the WifiCommunicator struct that allows the device to connect to WiFi networks and handle HTTP requests through a server. The module includes methods for setting up the WiFi driver, starting the connection, and registering request handlers.

The WifiCommunicator is essential for enabling network communication, allowing commands to be received and statuses to be reported back to external sources.
