firmware_core::printer

Module communication

Source
Expand description

This module contains everything related to making the 3D printer able to communicate with the external world. Communication methods may include:

  • WiFi
  • USB (not supported yet)
  • Bluetooth (not supported yet)

The data exchange with the external world is used to monitor the printer’s status and execute commands (like starting the printing of a 3D model).
