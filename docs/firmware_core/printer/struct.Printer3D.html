Struct Printer3D

pub struct Printer3D<P: Peripherals + 'static> { /* private fields */ }
Expand description

A structure representing a 3D printer.

This struct manages the printer’s components, communication protocols, and the watchdog timer for monitoring the printer’s operation. It provides methods to initialize the printer and perform periodic updates.

This method initializes the printer’s components and communication systems, setting up any necessary peripherals.

§Arguments

Updates the printer state, feeding the watchdog if present.
