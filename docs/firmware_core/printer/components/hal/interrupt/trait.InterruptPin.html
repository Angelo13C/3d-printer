Subscribes to an interrupt on this pin with a specified trigger condition.

§Safety

The callback will be called in an Interrupt Service Routine (ISR) context.

This means that the callback should be lightweight and should not perform operations that could block or take a long time. Care must also be taken to ensure that shared resources are accessed in a thread-safe manner.

Dyn Compatibility§

This trait is not dyn compatible.

In older versions of Rust, dyn compatibility was called "object safety", so this trait is not object safe.
