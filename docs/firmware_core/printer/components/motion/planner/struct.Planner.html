pub fn plan_move<K: Kinematics>( &mut self, target_position: VectorN<N>, steps_per_mm: [f32; N], move_speed_mm_s: f32, ) -> Result<MoveId, BlocksBufferIsFull>

Plans a move that will make the tool move to the target_position starting from the target_position of the last planned move.

steps_per_mm contains the number of steps required to move the tool by 1mm on each axis.

The planned move won’t be sent to the StepperMotorsTicker until you call Self::mark_last_added_move_as_ready_to_go.

Check Self for more info.
