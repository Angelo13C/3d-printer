firmware_core::printer::components::motion

Module planner

Source
Expand description

Translates movements (for example a 10mm move in the X axis) to motion profiles “understood” by stepper motors.

The generated motion profiles are optimized to make the move as fast as possible without provoking problems (for example making a stepper lose steps).
