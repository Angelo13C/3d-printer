pub struct StepperMotorsTicker<Timer: TimerTrait> { /* private fields */ }
Expand description

Attaches an interrupt to the timer you provide to Self::new that checks if stepper motors should take a step based on the planned block communicated to the ticker, and eventually makes them take the step.
