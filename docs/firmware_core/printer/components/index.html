firmware_core::printer

Module components

Source
Expand description

This module contains all the components necessary for the 3D printer to actually operate.

It includes various configurations, drivers, motion control, temperature management, and the overall control structures that enable the printer’s functionality.

§Key struct

  • Printer3DComponents: A struct representing the components of the 3D printer.

    G-code is the language understood by 3D printers, and it’s used to make the 3D printer execute commands (like move the tool to a specific location, set the bed’s temperature…).
  • hal
    Hardware Abstraction Layer (HAL) module.
  • mock
    This module provides mock implementations for each trait in the HAL (Hardware Abstraction Layer) module. These mock structs are useful for testing and simulation purposes, allowing developers to test functionality without requiring actual hardware.
  • motion
    This module is responsible for managing all aspects of motion control within the machine, including movement planning, homing, and bed leveling. It integrates various components like stepper motors, kinematics, and timers to facilitate precise movement.
  • pauser
    This module gives the ability to the printer to pause a print and resume it later.
  • print_process
    This module manages the process of printing files by parsing G-code commands.
  • temperature
    A module for managing temperature control in the 3D printer.
  • time
