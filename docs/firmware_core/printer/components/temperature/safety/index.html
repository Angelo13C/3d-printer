Module safety

Defines safety mechanisms to prevent overheating and ensure safe operation.

This module implements safety checks to ensure that the temperatures of the 3D printer’s hotend and heated bed remain within safe limits. It provides mechanisms to verify if the current and target temperatures are within specified allowable ranges and handles various temperature-related errors.
