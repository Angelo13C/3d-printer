Module bl_touch

Source
Expand description

This module provides the implementation for the BLTouch sensor, which is used for auto bed leveling in 3D printers.

The BLTouch sensor operates by using a servo motor to control a probe that can touch the printing bed. This module handles the commands to move the probe, as well as the interaction with the bed leveling process.
