where Self: Sized,

Check if the block at the provided block_index in the provided spi_flash_memory contains a bad block mark.

Returns Err(...) if there is a problem in reading from the flash memory and it has been impossible to check if the mark is there. Otherwise returns Ok(contains_bad_block_mark).
