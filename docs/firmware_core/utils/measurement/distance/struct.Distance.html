Struct Distance

Source 
pub struct Distance { /* private fields */ }
Expand description

A distance with a 10 nanometer sensitivity (10^-8 meters) and a range of values that goes from 10 * (-2^31) nm to 10 * (2^31-1) nm (which is from -21 to 21 meters).
